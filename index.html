Claw Orchestrator:
A Unified Runtime for Coding Agent CLIs

Guian Fang

Open-source, 2026

GitHub npm

A TypeScript runtime that turns interactive coding CLIs — Claude Code, Codex, Gemini, Cursor Agent, OpenCode, and custom engines — into persistent, programmable agents. Start long-running sessions, route tasks across engines, and coordinate multi-agent councils through one unified API. Runs standalone, with first-class OpenClaw plugin support.

PAI: Long-form Video Generation for Cinematic Storytelling

Utopai Studios*, Guian Fang*

Productized at Utopai Studios, 2026

A productized long-form video generation system for minute-level cinematic continuity. PAI maintains characters, tone, and narrative coherence across shots, with creator control from concept to final frame.

Launch post Product page

ACG Creation Made Easy for Everyone.

MikoAI*, Guian Fang*

Productized at MikoAI, 2025

arXiv Project page MikoAI

FramePrompt powers the character-animation pipeline behind MikoAI, an AI creation tool for anime, comic, and manga (ACG). It unifies reference images, skeleton-guided motion, and target video clips into a single visual sequence, reformulating animation as conditional future prediction — letting pre-trained video diffusion transformers drive controllable character animation with zero architectural changes, no explicit guider modules, and no multi-stage pipelines.

HumanRefiner:
Benchmarking Abnormal Human Generation and Refining with Coarse-to-fine Pose-Reversible Guidance

Guian Fang*, Wenbiao Yan*, Yuanfan Guo*, Jianhua Han, Zutao Jiang, Hang Xu, Shengcai Liao, Xiaodan Liang

ECCV, 2024

Dataset page Paper

A new large-scale benchmark (AbHuman) for anatomical anomalies in humans, and a plug-and-play method (HumanRefiner) for refining abnormal human generations with pose-reversible guidance.

RealignDiff:
Boosting Text-to-Image Diffusion Model with Coarse-to-fine Semantic Re-alignment

TNNLS, 2023

Paper
